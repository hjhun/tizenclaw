# tizen-sound-cli
**Description**: Control volume levels, list sound devices, and play tones.
## Subcommands
| Subcommand | Options |
|---|---|
| `volume get` | Get all volume levels |
| `volume set` | `--type <type> --level <N>` |
| `devices` | List sound devices |
| `tone` | `--name <TONE> --duration <ms>` |
