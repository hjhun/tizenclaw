# tizen-sensor-cli
**Description**: Read sensor data from device sensors.
## Usage
```
tizen-sensor-cli --type accelerometer|gravity|gyroscope|light|proximity|pressure|magnetic|orientation
```

## LLM Agent Instructions
**CRITICAL**: You MUST use the `--type` flag. Without `--type`, the command will fail.
Example: `--type accelerometer`
Do NOT use a subcommand like `accelerometer` without the `--type` flag.
