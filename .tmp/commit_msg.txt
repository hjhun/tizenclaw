Implement rebuilt Rust CLI surface

Add a new `rusty-claude-cli` crate as the production Rust CLI entry point
for the rebuilt workspace.

This change:
- adds modular CLI parsing, init, stdin merge, and rendering layers
- integrates runtime, commands, tools, and plugins into structured output
- keeps `tclaw-cli` as a thin compatibility wrapper over the new crate
- adds representative tests for parsing, defaults, output, and resume flows
